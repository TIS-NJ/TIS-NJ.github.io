## 开始写作

使用 Hexo 创建新文章非常简单：

```bash
hexo new post "文章标题"
```

然后在 `source/_posts/` 目录下找到新创建的文章文件，开始编辑吧！

## 特性

- 📝 Markdown 支持
- 🎨 美观的 Butterfly 主题
- 🚀 自动部署到 GitHub Pages
- 📱 完全响应式设计
